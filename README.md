# Simple_Banking_System
Written with:
  - the Luhn Algorithm
  - java.util.Scanner
  - Menus
  - Authorization
  - SQL scripts
  - prepared statements
  - SQLite database
  - StringBuilder

You can:
  - create accounts and then receive a correct card's number (the last digit is a verification code).
  - put money on a balance
  - transfer money between accounts
  - close accounts
